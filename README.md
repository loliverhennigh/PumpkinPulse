Pumpkin Pulse is a python package for scalable GPU accelerated simulations of plasma. The objective is to allow for full device, first principle modeling of a variety of fusion reactors such as Field Reversed Configuration, Z-Pinch, and Inertial Confinement Fusion machines. Key features include,

- Full 3D electromagnetic field solver using the Finite Difference Time Domain method, (2nd order in space, 2nd order in time ðŸŽƒ).
- GPU acceleration using NVIDIA Warp, (core kernels achieving 95% memory bandwidth utilization ðŸŽƒ).
- Scalable to multiple GPUs and nodes using MPI, (linear weak scaling proven up to 8 nodes ðŸŽƒ).
